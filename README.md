# Basic redis cache example

### Usage:
**OBS**: You need to have Redis running

```bash
yarn prisma migrate dev   
```

```bash
yarn prisma db seed  
```

```bash
yarn prisma studio 
```

**Docker:**

```docker
docker build -t image_name
```

```docker
docker run -p 3000:3000 image_name
```

**Node:**

```bash
yarn start:prod
```

or

```bash
yarn start:dev
```
